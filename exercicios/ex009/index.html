Listas em HTML

Listas ordenadas

Veja abaixo um exemplo de lista ordenada

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

A tag ol possui o parâmetro type onde podemos escoher o tipo de marcador atual, as opções para este parâmetro são: 1,A,a,I,i

Listas não ordenadas

Agora um exemplo de lista não ordenada

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

O marcador padrão é a bolinha preta totalmente preenchida, mas você pode usar o type e escolher entre: square, circle e disc

Listas mistas

Podemos tambem criar listas dentro de outras listas como no exemplo:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
    1. Item 3.1
    2. Item 3.2
    3. Item 3.3

Lista de definições

É como se fosse um dicionário, temos os termos e suas definições. É uma lisa sem demarcadores, mas bem útil em alguns casos.

Toda lista de definições está dentro da tag dl, cada item é um dt e cada descrição. é um dd. Essas duas últimas assim como itens da lista, essas duas últimas possuem fechamento opcional

Veja um exemplo de lista de definição:

CSS

JavaScript
Linguagem de programação para a criação de interatividades em sites
